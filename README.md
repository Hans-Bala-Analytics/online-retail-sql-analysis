# Online Retail Orders Analysis

## 📌 Project Overview
This project involved analyzing an online retail store's order 
management database using SQL. The goal was to answer key business 
questions that support data-driven decision-making for growth.

## 🎯 Objective
- Query and analyze order data to uncover sales and customer trends
- Support business decisions using structured data insights
- Validate data accuracy for reliable reporting

## 🛠️ Tools & Skills Used
- SQL (MySQL Workbench)
- Data Manipulation Language (DML)
- Joins
- Subqueries
- Built-in SQL Functions

## 🔍 Approach
1. Explored the "orders" database schema and table relationships
2. Wrote SQL queries using joins and subqueries to extract 
   meaningful business insights
3. Used built-in functions for aggregation, filtering, and 
   data transformation
4. Validated query outputs against business requirements

## ✅ Business Impact
The insights derived from these queries helped identify patterns 
in customer behavior and order trends, supporting decisions aimed 
at improving overall business growth.
